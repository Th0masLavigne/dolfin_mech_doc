dolfin_mech.Material_Elastic_Hooke

class HookeBulkElasticMaterial(kinematics, parameters)[source]

Bases: ElasticMaterial

Class implementing the spherical (bulk) part of the Hooke material model.

This model captures only the volumetric response, relating the pressure to the trace of the strain tensor.

The volumetric strain energy density is defined as:

\[\psi = \frac{d K}{2} (\text{tr } \boldsymbol{\varepsilon}_{sph})^2\]

where \(d\) is the spatial dimension and \(K\) is the bulk modulus.

class HookeDevElasticMaterial(kinematics, parameters)[source]

Bases: ElasticMaterial

Class implementing the deviatoric (shear) part of the Hooke material model.

This model captures the shape-changing response at constant volume, relating the deviatoric stress to the deviatoric strain.

The deviatoric strain energy density is defined as:

\[\psi = G \boldsymbol{\varepsilon}_{dev} : \boldsymbol{\varepsilon}_{dev}\]

where \(G\) is the shear modulus (equivalent to \(\mu\)).

class HookeElasticMaterial(kinematics, parameters)[source]

Bases: ElasticMaterial

Class implementing the classical isotropic linear elastic Hooke material model.

This model relates the Cauchy stress \(\boldsymbol{\sigma}\) to the infinitesimal strain \(\boldsymbol{\varepsilon}\) through the Lam parameters \(\lambda\) and \(\mu\).

The strain energy density \(\psi\) is defined as:

\[\psi = \frac{\lambda}{2} (\text{tr } \boldsymbol{\varepsilon})^2 + \mu \boldsymbol{\varepsilon} : \boldsymbol{\varepsilon}\]
